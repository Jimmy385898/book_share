# 社区图书共享平台 - 前端项目

## 项目结构

本项目已将用户端和管理员端合并为一个统一的前端应用。

### 目录结构

```
src/
├── layout/
│   ├── MainLayout.vue          # 用户端主布局
│   └── admin/
│       └── AdminLayout.vue     # 管理员端主布局
├── views/
│   ├── Login.vue               # 用户登录页
│   ├── Register.vue            # 用户注册页
│   ├── Home.vue                # 首页
│   ├── Books.vue               # 图书广场
│   ├── BookDetail.vue          # 图书详情
│   ├── PublishBook.vue         # 发布图书
│   ├── MyBorrow.vue            # 我的借阅
│   ├── Discussion.vue          # 讨论区
│   ├── DiscussionDetail.vue    # 讨论详情
│   ├── Rank.vue                # 排行榜
│   ├── Profile.vue             # 个人中心
│   └── admin/                  # 管理员页面
│       ├── AdminLogin.vue      # 管理员登录
│       ├── Dashboard.vue       # 数据统计
│       ├── Users.vue           # 用户管理
│       ├── AdminBooks.vue      # 图书管理
│       ├── Borrows.vue         # 借阅管理
│       ├── Reviews.vue         # 书评管理
│       └── Discussions.vue     # 讨论管理
├── router/
│   └── index.js                # 路由配置（包含用户和管理员路由）
├── stores/
│   └── user.js                 # 用户状态管理
└── utils/
    └── request.js              # HTTP请求封装
```

## 访问方式

### 用户端
- 登录页面：`http://localhost:5173/login`
- 注册页面：`http://localhost:5173/register`
- 首页：`http://localhost:5173/home`

### 管理员端
- 管理员登录：`http://localhost:5173/admin/login`
- 管理后台：`http://localhost:5173/admin/dashboard`

## 登录入口

在用户登录页面（`/login`）底部，点击"管理员登录"链接即可跳转到管理员登录页面。

## 路由保护

- 用户路由：未登录用户访问用户页面会自动跳转到 `/login`
- 管理员路由：
  - 未登录访问管理员页面会跳转到 `/admin/login`
  - 非管理员角色访问管理员页面会被拒绝
  - 管理员登录后自动跳转到 `/admin/dashboard`

## 启动项目

```bash
cd front/user
npm install
npm run dev
```

## 注意事项

1. 管理员账号需要在数据库中将用户的 `role` 字段设置为 `admin`
2. 用户和管理员使用同一个登录接口，通过 `role` 字段区分权限
3. 前端已删除原 `admin` 文件夹，所有功能集成在 `user` 文件夹中
